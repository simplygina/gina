Project Description:

Studio web maps represented weekly outputs in this web mapping course. Output 3 was a result of whatever style map I wished to create. This map features monthly July temperature data averaged over a decade. Users can select points showing a popup of temperature and elevation. Additionally, layers can be toggled hiding layers not selected.
