Project Description:

This project examines wildfire burn severity using the Google Earth Engine (GEE) JavaScript API. Specifically, this script calculates the Normalized Burn Ratio (NBR) to generate dNBR. Once dNBR is calculated, the resulting image is then classified into different degrees of burn severity: low severity, moderate-low severity, moderate-high severity, high severity. This analysis is based on the UN-SPIDER recommendations on best practices for evaluating burn severity. The script can easily be modified to any study area of interest by inputting a Landsat Path and Row, date pre-fire, and date post-fire. The script outputs an NBR, dNBR, and USGS-based Burn Severity Classification layer.
