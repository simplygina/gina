Project Description:

Building off of this Species Distribution Model (SDM) Tutorial in R, linked here, I predicted the species distribution for Eastern Hemlock (Tsuga canadensis) for present day and 2070. These layers were mapped with the slider tool in ArcGIS Online StoryMaps. The distribution of species presence data is also mapped here. More detail on this project is shown in the web mapping application.
